DogTag API Documentation

Endpoints

  • GET /dog - Get all dog entries from the database.

  • GET /dog/:id - Get single dog entry with the specified ID.

  • DELETE /plants/:id - Delete a plant with the specified id.

  • POST /dog - Create a new dog entry.

    • name is required

    breed, age, gender and housebroken not required


  • PUT /dog/:id - Replaces values for a dog entry specified by the id

    • name is required
